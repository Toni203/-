# Система управления записями пациентов ГКБ им. Пирогова

## Описание проекта
Веб-приложение для управления записями на приём к врачам в медицинском центре. 
Реализован полный цикл CRUD операций с хранением данных в localStorage.

## Функциональные возможности
-  Создание новой записи (ФИО, телефон, врач, дата, время)
-  Просмотр всех записей в виде карточек
-  Редактирование существующих записей
-  Удаление записей
-  Поиск записей по ФИО пациента
-  Фильтрация записей по врачу
-  Сортировка записей по дате и времени

## Технологии
- HTML5
- CSS3 (Flexbox, Grid, адаптивный дизайн)
- JavaScript (ES6+)
- LocalStorage для хранения данных
- Font Awesome и Google Fonts

## Как запустить
1. Клонировать репозиторий
2. Открыть `index.html` в браузере
3. Данные автоматически сохраняются в localStorage
