# Структура данных веб-приложения

## 1. Описание предметной области

Веб-приложение предназначено для управления записями пациентов на приём к врачам в медицинском центре "ГКБ им. Пирогова". Основная сущность приложения — **запись на приём (Appointment)**.

## 2. Объект Appointment

### 2.1 Структура объекта

| Поле | Тип данных | Обязательное | Описание | Пример значения |
|------|------------|--------------|----------|-----------------|
| `id` | `string` | +            | Уникальный идентификатор записи. Формируется из временной метки и случайной строки для гарантии уникальности | `"1743098765432abc123"` |
| `name` | `string` | +            | Полное ФИО пациента. Может содержать буквы, пробелы, дефис | `"Иванов Иван Иванович"` |
| `phone` | `string` | +            | Номер телефона в международном формате. Сохраняется в формате +7XXXXXXXXXX | `"+79161234567"` |
| `doctor` | `string` | +            | ФИО врача и его специализация. Выбирается из предопределённого списка | `"Петрова А.И. (кардиолог)"` |
| `date` | `string` | +            | Дата приёма в формате ISO (YYYY-MM-DD). Не может быть раньше текущей даты | `"2025-03-28"` |
| `time` | `string` | +            | Время приёма в формате HH:MM. Доступные часы: 09:00-17:00 с интервалом 1 час | `"14:00"` |

### 2.2 Обоснование выбора полей

| Поле | Обоснование |
|------|-------------|
| `id` | Необходим для уникальной идентификации каждой записи при операциях обновления и удаления. Использование комбинации timestamp + random исключает коллизии |
| `name` | Идентификация пациента. Необходимо для поиска записей и связи с медицинской картой |
| `phone` | Ключевое поле для связи с пациентом (подтверждение записи, напоминания, экстренная связь) |
| `doctor` | Определяет специалиста, к которому записан пациент. Позволяет фильтровать записи по врачу |
| `date` | Определяет день приёма. Необходима для сортировки и контроля актуальности записей |
| `time` | Определяет время приёма. В паре с датой формирует полную временную метку для сортировки |
